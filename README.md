# Pattern Incremental
Current version: 0.71 [Linear Update]

Inspired by Idle Dice, https://www.rngdle.com/, and Antimatter Dimensions (￣▽￣)

Pattern Incremental is a game all about rolling numbers for patterns. Each roll increments your point count, and it gets boosted based on how many patterns are in your number!
The more you play, the more digits and patterns you unlock. Autoclickers aren't necessarily encouraged but they're chill. There's no hardcap CPS or checks. If you want balanced gameplay, I wouldn't go above 8 ~ 10 CPS autoclicking.

**This game is VERY heavily reliant on RNG. Your progression will have both fast and slow moments. Please be patient!**

That being said, I do playtest it myself quite a bit to ensure a reasonable expected pace. Shoutout to the playtesters: glixh_hunt3r, the woke left, and a certain cruddy osu! 5 digit
